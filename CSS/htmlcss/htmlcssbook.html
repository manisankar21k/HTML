Block-level and Inline Elements

A block-level element starts on a new line and takes the full width. Example: <div>, <p>, <h1>.

An inline element does not start on a new line. It only takes the space needed. Example: <span>, <a>, <strong>.

Common Tags

<p>: Paragraph for text.

<h1> to <h6>: Headings, h1 is the biggest, h6 is the smallest.

<div>: Groups block elements.

<span>: Groups inline text.

Semantic Tags

Semantic tags give meaning to content. They tell what the content is, not just how it looks.

  • <header>: Top section of a page.
  • <nav>: Navigation links.
  • <main>: Main content.
  • <article>: Independent content like an article.
  • <aside>: Side content like notes.
  • <footer>: Bottom section.

HTML Forms

Forms are used to collect user input.

  • Input types: text, email, password, date, radio, checkbox.
  • Label: Tells what the input is for.
  • Placeholder: Shows a hint inside the box.
  • Required: Makes sure the user must fill it.
  • Validation: Browser checks the input (like correct email).

HTML Tables

Tables are used to show data in rows and columns.

<table>: Creates a table. <tr>: Table row. <th>: Table heading. <td>: Table data.

HTML Meta Tags and SEO

SEO means Search Engine Optimization. It helps search engines understand your page.

  • <meta charset>: Sets the character type (UTF-8).
  • <meta name="description">: Short summary of the page.
  • <meta name="keywords">: Important words about the page.
  • <meta name="author">: Page writer name.
  • <title>: Page title shown in the browser tab.

CSS Box Model

The box model is how every element is built.

  • Content: The text or image inside.
  • Padding: Space between content and border.
  • Border: Line around padding.
  • Margin: Space outside the border.

Flexbox Basics

Flexbox is used to arrange items easily.

  • display: flex; Turns on flexbox.
  • justify-content: Aligns items horizontally.
  • align-items: Aligns items vertically.
  • flex-direction: Row or column layout.
  • gap: Space between items.
